# HR Analytics Dashboard

## Project Overview
This HR Analytics Dashboard is developed using Power BI to analyze employee data and identify workforce trends. The dashboard provides insights into employee attrition, demographics, job satisfaction, and performance metrics.

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## Features
- Employee Attrition Analysis
- Department-wise Analysis
- Gender Distribution
- Age Group Analysis
- Job Satisfaction Analysis
- Interactive Filters and Slicers

## Dataset
The dataset contains employee information including:
- Age
- Gender
- Department
- Job Role
- Salary
- Attrition Status
